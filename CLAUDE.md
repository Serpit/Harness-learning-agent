## Project Overview

This is a knowledge repository containing a comprehensive Chinese-language technical article on **Agentic Engineering** — a software engineering paradigm where developers shift from writing code to coordinating AI agents. There is no application code, build system, or test suite.

## Content

The article covers the full lifecycle of adopting Agentic Engineering:

- **Concepts & cognitive shift** — from "Vibe Coding" to structured agent coordination
- **Context Engineering** — structuring information for AI decision-making (three-layer knowledge: long-term memory, overflow area, working memory)
- **System architecture** — 22 Agents, 27 Skills, 28 Commands organized in Team Intelligence / Platform Capability / Infrastructure layers
- **Design patterns** — Command vs Skill vs Subagent taxonomy, progressive disclosure, Ralph Loop (self-iterating agent execution)
- **Real-world case study** — operational activity configuration built entirely through AI agents
- **Team adoption** — scaling from individual to team-wide usage, knowledge accumulation strategies
